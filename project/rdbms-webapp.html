DATABASE MANAGEMENT SYSTEM FOR STOCK PHOTOGRAPHY WEBSITE

RDBMS & Web Application

Develop and design of database management system for stock photography website. The photographer’s photo distribution channel is often in a form of selling each photo with the buyer contacting the photographer directly to buy the photo. The photographer will record each transaction possibly in a document file format or paper form. In the traditional recording system, many problems occur such as misinformation and information loss due to a high amount of photos. From this issue, a database management system for stock photography is created by using the Oracle 11G program to manage the database with the PHP language in web application development. The application is connected to the database system. It is found that photographers can manage their own photos into categories. The total sales report or sales history report can be exported.
