HEALTH ME PLEASE

iOS Application

Healthy Me Please is an application on smartphone that a user could record his/her daily routine such as running, cycling, and calories burned. This application will be able to work with wearable devices like Fitbit, Pebble for tracking people’s activities record from their workout. The data from activities tracking will be stored in the application and could be shared to social network. There is also the challenge system, which a user can challenge other user to complete a healthcare mission and will receive unlocked badges as rewards if the mission complete, and earn the points to be ranked with other users.

Healthy Me Please developed in iOS. In parts of connection between application on smartphone and wearable, we will use libraries and API for wearable device to get a wearable device detected by a smartphone via Bluetooth. In the future, the application will be developed for more add-ons support and wearable devices compatibility to make it better because some wearable devices don’t support some smartphone’s operating system by now.
