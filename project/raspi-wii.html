MOBILE RASPBERRY PI REMOTE CONTROLLING

Embedded system

I have discuss about the project “Mobile Raspberry Pi Remote Controlling” with Dr. Vincent Guyot at ESIEA. To know what is Raspberry Pi and what it can do I have learn from raspberrypi.org site. First I learnt how to install “Raspbian OS” to work with Raspberry Pi. Raspbian is an unofficial of Debian wheezy and will run on the Raspberry Pi. I install Raspbian on the SD card after that plug the SD card into the Raspberry Pi board to use it.

I have learnt how to communicate between the Wii Remote and Raspberry Pi. At first, I receive bluetooth usb and Wii Remote from professor. The Wii Remote can connect to Raspberry Pi by bluetooth device. I setup the bluetooth on Raspberry Pi by plug it into the usb slot and install bluetooth package by this command line “sudo apt-get install –no-install-recommends bluetooth”. When the bluetooth package already installed in the Raspberry Pi, I can check the bluetooth status by type “sudo service bluetooth status” in command line. If bluetooth is working the terminal will show message “bluetooth is running”. To detect Wii remote on Raspberry Pi I have to type “hcitool scan” if Wii Remote was found it will show a message like “00:1E:02:8A:CD:A1 Nintendo RVL-CNT-01” on the screen. Before using Wii remote with raspberry pi I have to install CWIID MODULE by type “sudo apt-get install python-cwiid” and we have test its with the sample code on Raspberry Pi website.

For the last, I have to write the program in C & Python to control the robotics with Wiimote. The program will check the setupGPIO and then print the message to wait for a connection from Wiimote. If the connection succeeds the Wiimote will rumble and show the usage buttons message but during connection time out the program will quit. When Wiimote has connected successfully if the home button was pressed the program will start and the program will check buttons to control the directional movement of the robotic car. The program will work until the home button was pressed again.

The program show Wiimote is connected and how to use

You can use buttons or gyroscope to command the robot. Press button B to change mode between buttons and gyroscope
