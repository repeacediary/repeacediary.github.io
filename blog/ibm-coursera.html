Finally, I complete all course from IBM data engineering

Peace Rittha
20 Jun 2022

Here are the skills I have learnt from this course: Python programming, Data analysis, RDBMS, Shell scripting, Data warehousing, ETL/ELT, Data pipelines, BI, Big data, Machine Learning as shown in the course lists

Doing hands-on labs, I learn to code and use the tools such as Jupyter notebook for data analysis, MySQL & PostgreSQL for SQL, MongoDB for NoSQL, etc.
